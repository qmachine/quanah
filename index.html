Quanah

Quanah is a JavaScript library for writing concurrent programs in JavaScript that still run correctly in serial execution environments. QMachine, for example, uses Quanah extensively to implement a web service that acts as a supercomputer that uses web browsers as computational nodes. I am currently writing several papers about Quanah and applications that use it, but I promise I will write more documentation soon!

Because Quanah is written in pure JS, it can be used in both client- and server-side applications. Thus, this repository integrates with both Bower and NPM:

$ bower install quanah
$ npm install quanah

Quanah is also available through MinervaJS.

There are several "mirrors" available on Bitbucket, GitHub, and Google Code. Various project statistics are available on GitHub and Ohloh, too.
