Quanah

Quanah is a JavaScript (JS) library for writing concurrent programs that still run correctly in serial execution environments. QMachine, for example, uses Quanah extensively to build a World Wide Computer that can use web browsers as computational nodes. Currently, documentation is sparse because I am writing several papers and a PhD thesis about Quanah and applications that use it.

Because Quanah is written in pure JS, it can be used in both client- and server-side applications. Thus, this repository integrates with both Bower and NPM:

$ bower install quanah
$ npm install quanah

There are several "mirrors" available on Bitbucket, GitHub, and Google Code. Various project statistics are available on Codacy, GitHub and Open Hub, too.
